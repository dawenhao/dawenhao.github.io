# Shaow Mapping

Shddow Mapping是一种图像空间的算法。它有一个重要的思想，假如看到一个不在阴影里的点，那么就认为摄像机能够看到这个点，光源也可以看到这个点。经典的Shadow Mapping只能处理点光源，生成的阴影就是硬阴影。

# 点光源下如何生成阴影

  Shddow Mapping做了两步操作

  1. 先从光源看向场景，不做着色，只记录任何点看到的深度。
  2. 从相机处看场景，渲染出的点重新投影回光源，与之前记录的深度做比较。

# Shadow Mapping中的问题

  • 只能做硬阴影
  • 阴影的质量依赖于Shdow Map的分辨率，分辨率低的话会发生走样，产生锯齿
  • 浮点精度也会造成问题

# 硬阴影与软阴影

  软阴影其实是一种自然现象，物理上称作半影(Penumbra)，物理上对阴影做了两种区分，如果在一个地方完全看不到光源，这个地方叫做本影(Umbra)，可以部分看到阴影就是半影。
