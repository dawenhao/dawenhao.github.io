# 着色

本课程中着色是对不同的物体应用不同材质的过程。

# Blinn-Phong着色模型

Blinn-Phong是一种经验模型，所谓反射模型，就是光线如何与材质作用，如何去反射。通过观察下面的茶杯看光线是如何反射的。

  • 高光，颜色变化比较剧烈，比如有一个表面特别光滑，反射就会像镜面反射一样
  • 漫反射，颜色变化不是特别剧烈的
  • 环境光，接收不到光的一个点，被间接光照照亮

# 局部着色

  我们认为物体的一个局部的极小的范围内永远是一个平面，在平面任何一个点(shading point)上有：

  • 观测方向(Viewer direction, v)
  • 表面法线方向(Surface normal, n)
  • 光照方向(Light direction, l) 是从shading point指向光源
  • 表面参数(Surface parameters)
      以上三个向量都是单位向量，长度均为1。着色只考虑颜色，不考虑阴影。

# 漫反射(Diffuse Reflection)

  光线打到某个点上后，会被均匀的反射到各个方向去就是漫反射。漫反射跟观测方向没有关系。
  当物体表面(shading point)的朝向与光线有一定夹角，得到的明暗是不一样的。物体表面的法线与光线的夹角决定了这个物体被光照射下有多么亮。因为光是一种能量，那么物体表面能接收到多少能量跟物体表面周围的单位面积有关系。任何一个shading point周围的单位面积能接受到多少能量，就和法线与光线形成的角度有关系。
  兰伯特余弦定理：接受到的能量是与光照方向和法线方向的余弦是成正比的。

# 光照衰减

  光是一种能量，来自于一个点光源，在任何一个时刻，点光源辐射出来的能量（如果认为往四面八方的能量都是一样的）都在一个球壳上，在某一个时间能量在一个球壳上，在下一个时间它会在下一个球壳上。过了很长时间后，能量会在很远的一个球壳上，但是能量是守恒的，在真空环境中，能量传播是没有损失的，但是传播过程中时间越久，球壳越来越大，那么球壳上每一个点的能量就会越来越少。
  光线传播时间越长，单位面积接收到的能量就越少。假设知道一个点光源，又知道shading point离点光源有多远，在知道了有多少光传播到了shading ponit，又知道了有多少光会被shading point吸收，结合在一起就可以得到diffuse的表示方法了。

# 漫反射的表示

  假设有一个点光源，这个点光源距离shading point有一定的距离，这个距离是，这么远的距离传播到shading point的能量就是，而这些能量能被接收多少取决于法线与光照方向的夹角（兰伯特余弦定理），这样就可以算出关于漫反射能够看到多少能量。漫反射的明暗就可以计算出来了。

  1. 当点乘得到负数的时候，没有物理意义，这时光是照不到的，所以用此时认为是0。
  2. 之所以显示颜色，是这个点吸收一部分能量，反射一部分不吸收的能量，不同的点有不同的吸收率，那么就会产生不同的颜色，特别是对不同的波长。这里使用一个系数表示，如果系数是1，就说明所有能量都被反射了，会显示成白色，如果系数是0，就说明所有能量都被吸收了，显示黑色。这里应该使用三通道的一个向量来表示，这样可以表示各种颜色。

# 高光反射(specular)

  比较光滑的物体的反射都接近镜面反射，会沿着镜面反射方向有一个分布，当观察方向与镜面反射方向接近的时候，能够看到高光。
  当观察方向和镜面反射方向接近的时候，其实就是法线方向和半程向量很接近。所谓半程向量，如果我有一个入射方向和出射方向,可以求得角平分线方向，之后对角平分线方向做归一化就得到了半程向量方向。和接近一定程度上反映了和接近。Blinn-Phong模型做了这么一个假设，为了衡量能不能看到高光只要看和是否接近。
  如何衡量两个向量是否接近，很简单的办法就是点乘，如果两个向量比较接近那么结果就接近1，如果两个向量比较远，结果就接近0。这里镜面反射系数通常认为就是一个白色的。Blinn-Phong模型主要关注的是是否能看到高光，有多少能量被吸收就被简化掉了，所以这里没有和的点乘。
  之所以有个指数是因为，向量夹角余弦确实能够体现两个夹角是否接近，但是它容忍度太高了，比如当两个向量夹角在45°时，感觉已经离得很远了，但是两个夹角余弦仍然告诉我们它有个较大的值（值越大说明越近），仅仅用夹角余弦去生成高光的话会生成一个较大的高光这是不合理的。有了指数后，我们就可以认为只要夹角距离稍微远一点就不算作是高光了。实际操作中，指数大概会用到100到200的值。

# 环境光(ambient)

  在一些无法被直接光照照到的地方，理论上应该是完全看不到的也就是黑色，但是实际却是会有颜色的，这是因为有一些光线会被别的物体一直弹射，从四面八方打到任何的一个点上，所以无法被光直接照射的地方就接收到了来自环境的光。
  计算环境光照是一个非常复杂的事情，所以这里就做一个大胆的假设，任何一个点接收到来自环境的光永远都是相同的，光照强度就是，环境光也有颜色，就拥有一个颜色系数，把这两项结合在一起，就近似的得到一个环境光照。
  在Blinn-Phong光照模型中：

  1. 环境光照与直接光照方向没有关系。
  2. 环境光照与观测方向没有关系。
  3. 环境光照与法线也没有关系。
  4. 环境光照是一个常数。

# 着色频率

下图三个球，是完全相同的三个几何，为什么在着色后得到的结果各不相同呢？是因为着色频率造成的。

  • 逐面着色，每个面着色一次。
  • 逐顶点着色，每个顶点着色一次，顶点内部的颜色可以插值得出。
  • 逐像素着色，每个像素着色一次，每个像素都有一个法线方向。
      当几何的形状足够复杂的时候，就可以使用一些简单的着色从而得到更好的效果。

# 逐三角着色(flat shading)

  • 每个三角形都是一个平面，求出三角形法线，根据光照模型进行着色，三角形内颜色一致。

# 逐顶点着色(Gouraud shading)

  • 求每个顶点的法线，然后着色。
  • 内部颜色通过插值得出。

# 逐像素着色(Phong shading)

  • 每个像素都有一个法线，然后着色。
  • Phong shading和Blinn-Phong光照模型不是一回事。

# 逐顶点的法线

  任何一个顶点都和不同的三角形有关联，那么就认为这个顶点的法线是相邻面的法线求平均。但是有的三角形可能会很大，有的三角形可能会很小，所以这里是加权平均，则权就是三角形的面积。

# 逐像素的法线

  假如已经知道每个顶点的法线了，那么内部的法线使用重心坐标做插值，就可以得到两个顶点之间的所有法线了。重心坐标后面会提到。

# 重复使用纹理

  纹理是可以重复使用的，不同的位置可以使用相同的纹理。如果一个纹理设计的比较好，那么这个纹理在重复使用的时候，纹理之间可以无缝连接。

# 重心坐标(Barycentric Coordinates)

重心坐标主要是为了做在三角形内的插值。

# 为什么要在三角形内插值

  有很多操作是在三角形顶点上计算的，在三角形内部希望它们可以平滑的过渡，所以需要插值。

# 插值什么内容

  在三角形顶点上可以定义各种各样不同的属性，比如空间中的一个三角形应该对应纹理中的哪个三角形。当然还有其他属性，比如颜色、法线等等。

# 怎么做插值

  使用重心坐标。

# 应用重心坐标

  在了解了重心坐标后，就可以直接应用重心坐标来做三角形内部颜色的插值了。任何一个点，重心坐标告诉我们的是，如果我们要插值，那么插值的属性也应该要用重心坐标给它线性的组合出来。这个属性可以应用到各种目标上，比如位置、纹理坐标、颜色、法线等。

# 纹理放大

  纹理太小，在应用到一个高分辨率的目标上的时候，查询的时候会出现很多非整数的坐标，在应用时会把非整数四舍五入成整数（相当于一个纹素应用到了很多个像素上），就变糊了。为了解决这个问题就要解决当给一个非整数的坐标时，如何查询对应的纹素，就引出了双线性插值。

# 双线性插值(Bilinear)

  一个高分辨率的像素的中心映射到纹理上的一个非整数的位置上，所以这里想要知道这个非整数的位置上的值是多少。
  认为图二中非整数位置的那个点是点P，可以找到点P相临近的四个点，那么四个像素的水平距离就是s，竖直距离是t，这个s和t都是之间的，用s对点,点 以及点,点分别做一次线性插值，得到点和点。然后再做一次竖直上面的插值，用t对点和点进行插值。就可以得到点P的颜色了。
  双线性插值得到的效果依然不是最好的，还有双三次插值(Bicubic)，取周围的16个点，每次用四个做三个插值。

# 纹理缩小

  如果纹理太大，会引起更严重的问题。下图仅仅做简单的纹理查询的操作，近处会有严重的锯齿，远处出现了摩尔纹。
  造成这个这个问题的原因，在近处一个像素覆盖的纹理相对较小，在远处一个像素就能覆盖很大一片纹理。近处的像素可以用覆盖的纹理查询一下得到对应的颜色值，而远处的一个像素覆盖了纹理很大一块，那个点是无法代表那么大区域颜色的变化的。   可以使用超采样来解决这个问题，但是采样点需要很多，所以我们不希望这么做。那么如果我们不才样呢？所以这里引出了MipMap。

# 点查询和范围查询

  点查询很好理解，以纹理为例，给定一个点，它的值是多少，就是点查询。给定一个区域，我们立刻可以得到它内部的平均值（范围查询种类的其中之一），这个是范围查询。

# MipMap

  可以使用MipMap来做范围查询，它是一个近似的、快速的、方形的范围查询。"MipMap"这个词来自拉丁语，"Mip"的意思是很多不同的小的东西。对应图形学上来说就是一张图生成一系列图。
  比如说这里有一张纹理，我们把原始的纹理叫做第0层，那么我们可以生成更多更高层的纹理，使得每一层的纹理都是上一层的图缩小到一半（分辨率减半）。生成到最后还剩一个点，加入分辨率为128，那么一共就有层。
  MipMap的好处就在于有了纹理后在渲染之前先处理一遍，而且只比原纹理的存储增加了。

# 使用MipMap做范围查询

  我们要使用MipMap在一个正方形范围区域内做近似的范围查询，并且立刻要得到这个区域内的平均值是多少。任何一个像素都可以映射到纹理上的一个区域，那么怎么知道这个区域呢，首先我们取这个像素的中心和它相邻像素的中心，分别都投影到纹理上去， 这里可以做一个近似，这个像素到相邻像素的距离就是1，当这些像素分别映射到纹理上的时候，计算出它会占据多长的距离L。近似的得出一个像素在纹理上会得到一个多大的区域（图三）。
  当得到像素在纹理上的正方形后，通过，就能得到这个像素在第几层会对应一个像素，就直接查第层，就能得到对应的像素平均值。

# 三线性插值

  只查询某个层的对应的MipMap颜色平均值会造成两层连接处不连贯，这里就需要用到三线性插值来计算两层之间的过渡颜色值。这里先找到对应的两层，做两次线性插值，然后再把两个插值结果再做一次线性插值，就得到了两层之间的颜色平均值。一共做了三次插值，所以叫三线性插值。

# MipMap的缺陷

  MipMap只能查询方块的区域内，如果不是就会出问题了，由三线性插值带来的问题可以通过各向异性过滤解决。

# 更多的纹理应用

在现代GPU里我们认为纹理就是一块内存，以及我们可以对这块内存进行范围查询或者滤波。

# 环境光照(Environment Map)

  假如站在一个房间中，就会发现来自四面八方都会有光无论它是哪种光（直接光照、反射的光），如果我们可以把这些光记录下来，就是环境贴图或者说是环境光照。环境贴图是假设只记录光源的方向信息，认为它们无限远，没有深度。
  加入可以把环境光描述在某一个纹理上，那就可以用它来渲染一些东西。图二中认为，在屋子里放上一个非常光滑的金属球（像镜子一样的球），那么这个镜子反射出来的就是环境光，所以我们可以把环境光存在一个球上，然后把这张图展开，这就是Spherical Environment Map。

  为了解决球形环境贴图的问题，这里引入了立方体贴图。假设还是用一个球来记录来自任何方向的光线，不同的是我们认为这里有一个包围盒包围着球。从球心往任何一个方向打一道线直到它打到立方体的表面上，所以就是把光照信息存在一个立方体的表面上。所以这里会得到六张图（立方体有六个表面）。它会比球形环境光增加一些开销。

# 凹凸贴图

  纹理不仅可以描述颜色，它也可以定义任何不同位置任何不同的属性。所以这里可以定义任意一个点的相对高度（原本有一个基础的表面，这里定义的是沿着基础表面的法线方向向上向下各移动多少）是多少。   如图一，像橘子的东西，传统的想表示这么多凹凸不平的效果，会使用很多很多三角形，但是如果使用凹凸贴图就可以查询出来在三角形任意一个点它的相对高度是多少。这样就可以在不把几何形状变复杂的情况下应用一个复杂的纹理从而定义任何一个点它的相对的高度。相对的高度一变，法线就会随之发生改变，那么着色的结果也会相应的发生变化。
  通过凹凸贴图，我们可以定义一个复杂的纹理但是并不去改变任何的几何信息，而是把任何一个像素它的法线做一个扰动，通过定义不同位置的高度和它临近位置的高度差，重新计算法线。图二中黑色的就是原本的物体表面，黄色的是应用的凹凸贴图。如P点改变高度后的新法线是n，如何计算新的法线呢？
  如图三，假设任何一个点，原本几何是一个平面那么法线是向上的，在p点法线为，如果我们定义了一个凹凸贴图，那么可以求出在任意一个点上的梯度（切线、函数的导数），但是我们想要的是法线，而法线其实就是垂直与切线的一个方向，这里把切线逆时针旋转90°就是法线，所以在归一化。
  在实际的情况下（3D），可以求出u方向以及v方向的两个切线，算出切线后也可以求出新的法线（图四）
  凹凸贴图可以得到一个欺骗人的眼睛的一个效果，这里有个更现代化的做法，就是位移贴图，位移贴图会把三角形的顶点真的做一个位置的移动，应用位移贴图就要求模型足够细致。DX中还有一种技术叫动态曲面细分，这个技术不需要一开始就有一个足够精细的模型。
