# 几何的归类

几何分为隐式几何和显示几何两种。使用哪种表示方法要根据实际需要。

# 隐式几何表述(Implicit)

  要表述一个几何的面，这里不会表示这些点具体在哪，而是表示这些点满足的关系，比如满足一定特定关系的点都认为在一个表面上。也就是说把这些点做一个归类。
  如图一，隐式几何可以通过一个函数找出这个几何在一个表面上的所有的点，有了这些点就可以把这个几何画出来。
  想要知道哪些点是在一个隐式几何上的，这是一个相对困难的事情。如图二，从一个式子上很难看出这是一个圆环。但是隐含的几何也有好处，比如判断一个点是不是在这个几何上是一个很简单的事情，只需要把这个点带入到关系式里，会得到一个正的或者负的或者是0的一个值，如果结果是0我们就认为这个点在表面上，如果是负的就认为这个点在物体内，如果是正的就认为这个点在物体外。

# 显示几何表述(Explicit)

  显示的几何表示要么直接给出顶点，要么通过参数映射的方式给出。
  三角形面就是一个典型显示几何的表示方法。
  通过参数映射的方法定义的表面，如图一，定义了空间和，上面有任意的点，并且定义了每个点，都会对应空间中的某一个点，即给定一个函数输入输出的是，把所有的遍历一遍，就能找到所有对应的。
  对于显示的几何表示，想要判断某个点是否在表面上，就比较困难。

# 隐式几何

对于一些简单的几何使用隐式方式表述的非常到位，但是对于复杂的几何就会很困难。
  CSG图二，是通过一些基本几何的基本运算(bool运算)来定义一个新的几何，就可以得到一些复杂的几何。
  距离函数图三，对于任何一个几何都不直接去描述表面，而是描述任何一个点到这个表面的最近距离。通过对几何的距离函数做融合可以形成图三的效果。所谓距离函数，是指空间中的任何一个点到你想要表述的这个几何形体上面的任意一个点它们之间的最小距离（可以是正的也可以是负的）。
  图四中，输入（A）和输出（B）是两张不同的图，斜线部分是一个物体挡住了视口的一部分，在A中挡住了三分之一，经过一定的移动后挡住了B中的三分之二，这里希望求得这个物体从左至右（从A到B）的移动的中间状态，那么实际得到的中间状态左边是被挡住的而中间部分只能被挡住一半（灰色，黑白的融合），把两张图做这样一个线性融合，并不能表述这个物体的运动信息，因为我们希望得到的结果是左边一半是黑的，右边一半是白的。
  怎么才能做对呢？首先对A求得一个距离函数，同时对B也做距离函数，所谓距离函数就是任何一个点到这个边的最短距离，这里就是对这条边做垂线，然后把这两张图做一个融合操作，那么正中间的一定是个0，左边是负的右边是正的。0的地方是边界，非0的地方不是。

# 分型

  分型指自相似，意思是物体的整体和它的一部分非常相似，有点类似计算机中的递归。

# 显示几何

显示几何的表示方法有很多种。

  • 三角形面
  • 贝塞尔曲面
  • 细分曲面
  • 点云

# 点云

  点云不考虑物体是一个表面，而是表面上的一堆点，只要这些点足够细，自然就看不到点与点之间的缝隙，一个点就是空间中的一个。如果点比较稀疏，就看不出是一个面。一般扫描物体后会得到一个点云，而后会把点云转化成多边形面。

# 多边形面

  在图形学中得到了非常广泛的应用，特别是三角形和四边形面。

# 贝塞尔曲线

  贝塞尔曲线，是一种显示的几何表示方法。它是要用一系列的控制点来定义某一个曲线。我们要求贝塞尔曲线要经过控制点的起止点。GAMES101 P11 13:01。

# 二次贝塞尔

  有三个控制点的贝塞尔曲线叫做二次被贝塞尔曲线。这条线一定是从b0开始b2结束，而b1就决定了这条曲线怎么弯曲。假设这条曲线它的起点在时间0，终点为时间1，那么给定一个时间t就可以找到一个点。在每一段都能找到对应的点，然后如果有多个点就连起来继续往下找在时间t的时候得到的点。

# 三次贝塞尔

  三次贝塞尔与二次贝塞尔操作一致，只不过控制点增加了。

# 贝塞尔曲线的性质

  贝塞尔曲线有一些不错的性质。

  • 规定必须过起点和终点，t=0的时候一定在起点，t=1的时候一定在终点。
  • 三次贝塞尔曲线（四个控制点），起始的切线方向和终止切线方向一致都是3。
  • 仿射变换下直接对顶点做仿射变换，直接对变换后的顶点画一条贝塞尔。
  • 凸包性质

# 细分

说到细分第一反应是引入更多的三角形，而细分实际做了两件事，第一，在原三角形上分出更多的三角形。第二，让三角形的位置发生变化，使得原来的模型变得更加的光滑。

# Loop细分（只能对三角形做细分）

  对于loop细分，一个三角形连接三角形的三个中点，最终得到四个三角形。这里提示一点，loop细分不可以理解为循环细分，这个名字是因为发明这个细分的人姓loop，和循环没有任何关系。
  loop细分会把新的顶点和旧的顶点区分开，用不同的方法去修改新旧顶点的位置。如图一
  那么如何调整这些三角形顶点的位置呢？如图二，对于新的顶点，首先新的顶点一定在一条边上，只要这条边表示的不是物体的边界，那么它一定会被不同的三角形共享，把共享的两条边叫AB，三角形不共享的顶点叫做CD。Loop细分的规则是，3/8的A+B加上1/8的C+D。得到新的白点。
  对于旧的顶点，如图三，首先定义一个顶点的度，顶点的度就是顶点连接的边的数量，再定义一个u。

# Catmull-Clark细分(可以做普通的mesh)

  首先定义四边形面(quad face)和非四边形面(Non-quad face)，然后定义一个奇异点(Extraordinary vertex)，奇异点即是度不为4。要做细分肯定要引入更多的点和边然后调整它们的位置。首先每条边都取中点，每个面也取中点，然后把边上的中点和面上的中点连起来，如图二。
  Catmull-Clark细分在细分之前如果有多少个非四边形面在一次细分之后都会变成多少个奇异点。没有非四边形面后再做细分不会增加新的奇异点。
