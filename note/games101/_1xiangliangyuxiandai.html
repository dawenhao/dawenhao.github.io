# 点积

  两个向量进行点积得到的是一个数量。可以帮助大家快速得到两个向量的夹角(例如)，特别是在两个向量都只是方向（即单位向量时）。
  对点积的结果进行反余弦，就是两个向量之间的夹角（单位为弧度）。

  点积满足交换律、结合律、分配律。

  在笛卡尔坐标系下，点积的运算更加简单。对应的元素相乘，然后再加起来，就得到点积的结果。

# 点积在图形学上的作用

  点积还可以用来找到一个向量投影到另一个向量。 投影到 ，在上的投影肯定是沿着方向,

  计算出投影，可以把一个向量分解成两个向量，其中一个方向平行于某个方向，另一个方向垂直于某个方向。比如已知的计算，另一个方向就可以计算出来。

  点积可以得到前与后的信息，如果有个向量的终点落在虚线以上，我们认为基本是相同方向，或者说都是向前的。而的终点落在了虚线以下，我们认为方向基本是相反的。其中,如果有一个向量与点积结果等于0，则这个向量在虚线上即两个向量垂直。

  点积还可以计算两个向量在方向上有多么接近，让两个向量进行点积，如果两个向量比较接近，那么点积结果就会比较接近1，假如说这两个向量逐渐远离，那么它们的点积结果就会逐渐变小，当这两个向量逐渐垂直，那么它们的点积结果也会逐渐变为0。直到它们完全相反，点积的结果就会变成-1。

# 叉积

  叉积虽然也是一种乘法，但是它和点积完全不一样的计算。它会输入两个不同的向量，它会给出另外一个向量。新算出的向量与原本的两个向量都要垂直。假如新向量是,要垂直于，又要垂直于，那么一定垂直于所在的平面。也就说明了一定和不在一个平面内。

  想要知道叉积得到的向量的方向，需要用到右手螺旋定则：四指的方向代表了旋转的方向，要表示和的叉积，就把四指从旋转的方向，拇指对应的方向就是叉积出来结果的向量方向。同时这也说明向量的叉积不满足交换律（如果要交换顺序一定要加一个负号）。
  如果在一个三维坐标系中，轴叉积轴得到轴，那么我们就说这个坐标系是个右手坐标系（需要用右手螺旋定则）。一个向量叉积它自己，得到的是0向量。

  叉积在代数上的运算稍为复杂，同时叉积也可以转换成矩阵(dual matrix)来进行计算。

# 矩阵

  矩阵就是一堆数，把他们安放在一个有几行几列的结构里。

# 矩阵的乘积

  可以乘的矩阵必须是前一个矩阵的列与后一个矩阵的行数相等。（M*N)(N*P) = (M*P)。
  假如说想知道结果的2行4列的结果，那么在第一矩阵中找第2行，第二个矩阵中找第4列，找到这两组数据，求一个点积就就可以了。

矩阵的乘积没有任何交换律！！！但是有结合律和分配律。

  矩阵与向量相乘，那么向量需要在左边并且是个列向量(m * 1的一个矩阵)，矩阵在右边。

# 转置

  矩阵的转置就是把矩阵的行和列交换。

# 单位矩阵

  单位矩阵就是一个对角阵，所谓对角阵就是只有对角线上有非零的元素。这里的对角线上全部都是1.
  如果有一个矩阵，乘上你的矩阵得到一个单位矩阵，那么就说这个矩阵是你的矩阵的逆，也说这两个矩阵是互逆的。
